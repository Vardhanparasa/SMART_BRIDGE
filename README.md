# AI-Enabled Student Admissions & Support Management System

## Project Overview

This project is developed as part of the SmartBridge Salesforce Internship.

The system helps educational institutions manage student admissions using Artificial Intelligence and Salesforce concepts.

## Features

- Student Admission Form
- AI Priority Score Prediction
- Admission Risk Analysis
- Smart Recommendation System
- AI Summary Generation
- Salesforce-based Admission Workflow

## Technologies Used

- Python
- Flask
- Machine Learning (Scikit-learn)
- HTML
- CSS
- Bootstrap
- Salesforce CRM
- GitHub
- Render
